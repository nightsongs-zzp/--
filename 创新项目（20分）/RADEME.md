# 🌍 太阳系模拟器 - Solar System Simulator

一个基于物理定律的交互式太阳系模拟器，使用Python和Matplotlib构建。

![模拟器演示](https://img.shields.io/badge/状态-运行中-brightgreen)
![Python版本](https://img.shields.io/badge/Python-3.7+-blue)
![许可证](https://img.shields.io/badge/许可证-MIT-green)

## ✨ 功能特色

- 🪐 **完整的太阳系模型**：包含太阳和八大行星
- 🎮 **交互式控制**：滑块实时调节参数
- 📊 **物理数据显示**：实时显示距离、速度等物理量
- 🎨 **精美可视化**：彩色编码的行星和轨道轨迹

## 🎛️ 控制面板

### 1. 时间速度 (Time Speed)
- **范围**: 0.1x - 5.0x
- **功能**: 控制模拟时间的流逝速度
- **效果**: 值越大，行星运动越快

### 2. 重力常数 (Gravity)
- **范围**: 0.5G - 2.0G
- **功能**: 调节引力强度
- **物理效应**: 改变行星的轨道速度

### 3. 显示轨道 (Show Orbits)
- **类型**: 复选框
- **功能**: 切换行星轨迹显示
- **教育价值**: 直观展示行星运动路径

### 4. 显示物理信息 (Show Physics)
- **类型**: 复选框
- **功能**: 显示实时物理参数
- **显示内容**: 距离、速度、模拟参数

## 📚 物理原理

### 演示的物理定律
1. **开普勒三大定律**
   - 椭圆轨道（简化圆形）
   - 面积定律
   - 调和定律

2. **牛顿万有引力定律**
   - 通过重力参数模拟
   - 距离平方反比关系

3. **圆周运动定律**
   - 向心力与轨道速度
   - 角速度与周期关系

### 计算公式
```python
# 行星位置计算
angle = 2 * π * time / period
x = radius * cos(angle)
y = radius * sin(angle)

# 轨道速度
orbital_speed = 2 * π * orbital_radius / period
```

## 🖥️ 界面说明

### 主视图区域
- **中心**: 太阳（固定不动）
- **轨道环**: 行星运行轨迹（虚线）
- **坐标网格**: 天文单位(AU)坐标系统
- **比例**: 1单位 = 1AU（日地距离）

### 信息显示区
- **左上角**: 物理参数和模拟状态
- **右上角**: 行星颜色图例
- **标题**: 当前模拟时间
