使用说明
环境要求
Python 3.6+
numpy
matplotlib
ipywidgets
安装依赖
pip install numpy matplotlib ipywidgets

运行方式
Jupyter Notebook:
将代码复制到Jupyter Notebook单元格中
运行单元格，即可看到交互式控件和图形
Jupyter Lab:
确保已安装ipywidgets扩展：jupyter labextension install @jupyter-widgets/jupyterlab-manager
运行代码
Google Colab:
直接复制代码到Colab单元格运行
可能需要安装依赖：!pip install ipywidgets

交互控件
波1振幅：控制第一个波的振幅（0.1-2.0）
波2振幅：控制第二个波的振幅（0.1-2.0）
相位差：控制两个波之间的相位差（0-2π弧度）

功能特点
实时可视化：
上部图：显示两个原始波和它们的叠加波
下部图：显示干涉强度分布
自动标记相长和相消干涉区域
物理原理说明：
实时显示当前参数的物理分析
解释干涉条件
计算理论干涉极值

教学价值：
直观展示相长干涉（红色标记）和相消干涉（蓝色标记）
清晰显示振幅和相位差对干涉图案的影响
